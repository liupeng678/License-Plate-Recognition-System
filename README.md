# 完成对视频中的多个车牌的识别


## 1.下载代码
由于视频文件过大，需要从https://pan.baidu.com/s/1IUtWsmfCuuFQFk9YkReG1g 提取码：83zy  下载视频，将视频放在最前目录下（和svm.dat同级）
## 2.搭建运行环境 
pip install -r requirements.txt
## 3.运行main函数
python main.py

得到博客https://blog.csdn.net/liupeng19970119/article/details/105595961
